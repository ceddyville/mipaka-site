Mipaka API

Normalized, machine-readable administrative division data for 7 East & Central African countries. From regions down to villages, plus historical eras and place names. Open source, community-driven.

Base URL https://api.mipaka.dev/api/v1
Open API — All endpoints are read-only and require no authentication. Rate limited to 1,000 requests/day for anonymous traffic. Unlimited via RapidAPI.
Contributions — The POST /contributions/ endpoint accepts community submissions (new divisions, corrections, coordinates). All submissions go through admin review before going live.
Countries

Country records include ISO codes, division levels, and historical eras. The primary entry point for exploring a country's administrative structure.

GET /countries/ List all countries with division levels & eras
Response Fields
code string ISO 3166-1 alpha-2 country code (e.g. KE)
name string Country name in English
division_levels array Ordered list of level definitions (name, level number)
eras array Historical eras (era_type, year_start, year_end, colonial_power)

Divisions

The core resource. Divisions form a self-referential tree — each division has an optional parent, creating hierarchies of unlimited depth. Filter by country, level, parent, or search by name.

GET /divisions/ List & filter divisions
Query parameters
countryoptional string Filter by ISO code. e.g. ?country=KE
leveloptional integer Filter by level number (1 = top-level)
parentoptional integer Filter by parent division ID
qoptional string Full-text search on division name
yearoptional integer Return names as they were in a given year
nameoptional string Match by historical name (e.g. ?name=Stanleyville)
curl 
curl https://api.mipaka.dev/api/v1/divisions/?country=KE&level=1
curl https://api.mipaka.dev/api/v1/divisions/?q=nairobi
curl https://api.mipaka.dev/api/v1/divisions/?name=Léopoldville
curl https://api.mipaka.dev/api/v1/divisions/?country=UG&level=6 // all villages

Contributions

Community-contributed data goes through admin review. Submit new divisions, corrections, coordinates, or historical names.

POST /contributions/ Submit a contribution for review
Request body
typerequired string new_division · correction · coordinates · historical_name
division_idoptional integer Existing division to correct or add data to
datarequired object Contribution payload (varies by type)
contributor_nameoptional string Name for attribution (shown if approved)
contributor_emailoptional string Get notified when your contribution is approved
curl · submit new division 
curl -X POST https://api.mipaka.dev/api/v1/contributions/ \
  -H "Content-Type: application/json" \

Admin review — All contributions are queued for review. You'll receive an email notification when your submission is approved or if we need more information.
Integration Guide

Drop Mipaka into your app in minutes. Here are examples for common frameworks — all endpoints return JSON with consistent pagination.

⚡ JavaScript / Fetch
app.js 
const API = "https://api.mipaka.dev/api/v1";

// Get all Kenyan counties
const counties = await fetch(`${API}/divisions/?country=KE&level=1`)
  .then(r => r.json());

// Build a cascading dropdown
async function loadChildren(parentId) {
  const res = await fetch(`${API}/divisions/${parentId}/children/`);
  return res.json();
}
🐍 Python / Requests
mipaka.py 
import requests

API = "https://api.mipaka.dev/api/v1"

# All Uganda villages (paginated)
resp = requests.get(f"{API}/divisions/", params={
    "country": "UG",
    "level": 6,
    "page": 1
})
villages = resp.json()["results"]

# Find a place by its colonial name
result = requests.get(f"{API}/divisions/", params={
    "name": "Léopoldville"
}).json()
⚛️ React / Next.js
CountyPicker.tsx 
import { useState, useEffect } from "react";

const API = "https://api.mipaka.dev/api/v1";

export default function CountyPicker() {
  const [counties, setCounties] = useState([]);
  const [wards, setWards] = useState([]);

  useEffect(() => {
    fetch(`${API}/countries/KE/top/`)
      .then(r => r.json())
      .then(setCounties);
  }, []);

  const onSelect = async (countyId) => {
    const data = await fetch(
      `${API}/divisions/${countyId}/children/`
    ).then(r => r.json());
    setWards(data.results);
  };
  // ... render dropdowns
}
🎯 Flutter / Dart
mipaka_service.dart 
import 'package:http/http.dart' as http;
import 'dart:convert';

class MipakaService {
  static const _base = "https://api.mipaka.dev/api/v1";

  Future<List> getCounties(String country) async {
    final res = await http.get(
      Uri.parse("$_base/countries/$country/top/"),
    );
    return jsonDecode(res.body);
  }

  Future<Map> getDivision(int id) async {
    final res = await http.get(
      Uri.parse("$_base/divisions/$id/"),
    );
    return jsonDecode(res.body);
  }
}
Common use cases
📍 Address forms
Cascading Country → Region → District → Ward dropdowns for address collection. Use /countries/{code}/top/ then /divisions/{id}/children/.
📊 Data dashboards
Map-based visualizations with region-level aggregation. Use /divisions/?country=KE&level=1 to get all counties for a choropleth.
🔍 Search autocomplete
Type-ahead location search. Use /divisions/?q=nai for fuzzy matching. Results include level and country context.
🏛️ Historical research
Trace how places were named in different eras. Use /divisions/?year=1920 or /names/?name_type=indigenous.
Pagination — All list endpoints return paginated results (default 50 per page). Use ?page=2 to navigate. Response includes count, next, and previous for easy traversal.

🇰🇪 Kenya · KE
Kenya
Jamhuri ya Kenya
2,303 divisions 3 levels deep 5 historical eras 516 historical divisions
Administrative Hierarchy
County → Constituency → Ward
Kenya has 47 counties established by the 2010 Constitution, replacing the former 8 provinces. Each county contains constituencies, which contain wards.
Brief History

Kenya's administrative boundaries have been redrawn more than any other system in East Africa. Under British colonial rule from 1920, the territory was divided into provinces — a structure that persisted for nearly a century through independence in 1963.

The 2010 Constitution abolished the provincial system entirely, replacing 8 provinces with 47 counties — each with an elected governor and county assembly. This devolution was the most radical administrative restructuring in Kenya's history, designed to bring government closer to the people.

Today each county is subdivided into constituencies (for parliamentary representation) and wards (the smallest electoral and administrative unit), as defined by the Independent Electoral and Boundaries Commission (IEBC).

Nairobi

Hierarchy
Kenya → Nairobi County → 17 Constituencies → 85 Wards
Brief History

The area now known as Nairobi was originally a Maasai watering hole called Enkare Nairobi — "the place of cool waters" in the Maa language. The Maasai pastoralists used the swampy grasslands along the Nairobi River as a seasonal grazing ground for centuries.

In 1899, the British East Africa Protectorate established a railway depot here as the Uganda Railway pushed inland from Mombasa. The depot quickly grew into a supply station, and by 1907 Nairobi had replaced Mombasa as the colonial capital — a role it has held ever since.

After independence in 1963, Nairobi became the capital of the Republic of Kenya. The 2010 Constitution created 47 counties, with Nairobi designated as both a county and the national capital — the only county with this dual status. Today it is divided into 17 constituencies and 85 wards.

🇺🇬 Uganda · UG
Uganda
Jamhuri ya Uganda
83,017 divisions 6 levels deep 4 historical eras 5 traditional kingdoms
Administrative Hierarchy
Region → District → County → Sub-county → Parish → Village
Uganda has the deepest administrative hierarchy in East Africa — 6 levels from region down to village, with over 83,000 divisions total.
Brief History

Before colonisation, the territory now called Uganda was home to powerful kingdoms — Buganda, Bunyoro, Toro, Ankole, and Busoga — each with their own governance structures. The Kingdom of Buganda, from which the country takes its name, was the most centralised.

The British declared the Uganda Protectorate in 1894, gradually absorbing the kingdoms into a colonial administrative system of provinces and districts. After independence in 1962, the kingdoms initially retained ceremonial power — until President Obote abolished them in 1967.

The kingdoms were restored as cultural institutions in 1993 under Museveni, but without political authority. Today Uganda's administrative structure runs six levels deep: 4 regions → 135+ districts → counties → sub-counties → parishes → villages, making it the most granular system in the region.
